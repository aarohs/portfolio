Net Present Value Website Calculator

Web Design and Development UI/UX

This browser-based Net Present Value calculator was developed for the CO1109 Business and Financial Computing module at the University of Leicester, where I achieved a grade of 100% for this assignment.

  • Built using JavaScript, HTML, and CSS with dynamic input handling
  • Allows users to create and compare multiple project cash flow scenarios
  • Calculates Net Present Value (NPV) using user-defined cash flows and discount factors
  • Displays real-time Present Value calculations per year and totals per project
  • Highlights the best project based on the highest NPV
  • Styled with a dark theme and intuitive button interactions for ease of use

The tool blends financial logic with interactive design to provide a clear, responsive user experience for evaluating project profitability.

VR Exposure Therapy – Group Project Using WebXR

UI/UX Web Design and Development Digital Media

As part of the Entertainment Technology module, our group project explored Extended Reality (XR) - covering VR, AR, and MR - and focused on designing and implementing a virtual world using WebXR. Through domain analysis, solution design, prototype implementation, and evaluation planning, we tackled the specific phobia of Acrophobia (Fear of heights) within an immersive environment.

  • Performed domain analysis of XR technologies (VR, AR, MR) and selected Exposure Therapy as our application focus
  • Designed a WebXR prototype using A-Frame and Three.js to create a VR Glass Elevator Exposure Therapy
  • Implemented interactive UI/UX flows for safe, progressive exposure sessions, with real-time progress tracking and adaptive difficulty
  • Collaborated via GitHub across a group of 5 to integrate 3D assets and ensure cross‑platform support across different browsers
  • Sketched an evaluation plan to test therapeutic efficacy, comfort, and user engagement metrics in future user studies alongside our project report

This project demonstrates the fusion of front‑end web development, immersive digital media, and therapeutic design to deliver browser‑based VR experiences for mental health applications.
